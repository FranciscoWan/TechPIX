# Landing Page Fintech PIX

Uma landing page moderna para uma fintech de pagamentos via PIX, desenvolvida com foco em experiência visual, animações fluidas e interface dinâmica.

O projeto apresenta uma interface responsiva com elementos animados utilizando GSAP, simulando funcionalidades comuns de plataformas financeiras digitais, como dashboards, métricas em tempo real, cartões informativos e interações suaves entre seções.

---

## Funcionalidades

- Animações avançadas com GSAP
- Transições suaves entre seções
- Interface responsiva
- Dashboard financeiro estilizado
- Cards animados
- Contadores e elementos dinâmicos
- Layout moderno inspirado em fintechs

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GSAP
